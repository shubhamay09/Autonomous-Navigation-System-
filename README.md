# 🚗 AI-Based Autonomous Navigation System

## 📌 Overview
This project demonstrates an AI-based autonomous navigation system capable of navigating a grid-based environment with dynamic obstacles using the A* path planning algorithm.

The system continuously adapts its path in real-time and tracks performance metrics such as path length, execution time, and replanning frequency.

---

## 🎯 Features
- Dynamic obstacle handling
- Real-time path replanning (A*)
- Smooth agent movement
- Interactive goal selection (mouse click)
- Performance metrics tracking:
  - Path Length
  - Time Taken
  - Session Replans
  - Total Replans
- Clean visualization with UI panel

---

## 🧠 System Workflow
1. Environment generation (grid + obstacles)
2. Path planning using A*
3. Dynamic obstacle movement
4. Replanning when path is blocked
5. Navigation execution
6. Performance monitoring
